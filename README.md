# SmartChat

## Environment Variables

You can control the website through environment variables.

-   API_KEY
-   MODEL
-   AUTH_SECRET
-   AUTH_GITHUB_ID
-   AUTH_GITHUB_SECRET
-   KV_URL
-   KV_REST_API_URL
-   KV_REST_API_TOKEN
-   KV_REST_API_READ_ONLY_TOKEN
